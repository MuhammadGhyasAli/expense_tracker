# Expense Tracker

A CLI expense tracker that accumulates expenses and displays a summary. Enter amounts one at a time; the program shows a running total and final statistics.

## Features

- Enter expense amounts one at a time
- Running total after each entry
- Final summary: total spent, count, average, min, max
- Input validation: rejects non-numeric, negative, and zero values

## How to Run

```bash
python expense_tracker.py
```

## Usage

Type a positive number to log an expense. Type `done` to finish and see the summary.
